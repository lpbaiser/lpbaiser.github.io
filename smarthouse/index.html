Get Started

Projeto

Este trabalho tem por finalidade a construção de um sistema de automação residencial, neste projeto fizemos a implementação de uma porta, portão e alarme acionados por comandos. Um ventilador acionado acionado automáticamente e manualmente por um sensor de temperatura. Os comandos e sinais partem de um servidor que transfere por meio de bibliotecas os comandos, acionando os atuadores acoplados ao arduino.

Hardware

Nesta seção veremos os componentes utilizados para construção do hardware. Veja a lista dos materiais necessários:
  • 1 Arduino UNO
  • 1 Protoboard
  • 1 Cabo USB
  • 2 Servo Motores
  • 1 Sensor PIR de presença
  • 1 Buzzer
  • 1 Fan
  • 1 Sensor de temperatura LM35
  • Jumpers

How to use

- Dependências

Nós utilizamos na codificação do projeto a linguagem JavaScript com servidor Http Node.js e a biblioteca Johnny-Five. Johnny-Five é uma biblioteca para desenvolvimento robótico e IOT, ela integra javascript com inúmeras plataformas de baseadas no Arduino e não arduino.

- Let's go

  • Clone o projeto disponível no Github
  • Navegue até a pasta do projeto clonado
  • Compile o arquivo Standard Firmata na IDE do arduino e exporte para a placa
  • Instale o NPM
  • Instale o NODE
  • Crie uma base de dados Mysql local com o nome: bdarduino
  • Instale as dependências do projeto executando: npm install
  • Rode o arquivo create_tables.js, no terminal com o comando: npm create_tables.js para criar as tabelas na base de dados
  • Rode o servidor com o comando: npm run dev
  • Abra o seu navegar no endereço: http://localhost:3000
  • Login e senha padrões são: admin
  • Após o login uma página será exibida com os controles disponíveis para interação com o Hardware
